ANCODE
-Se necesita node js y mongodb.

Pasos a seguir:
1.Abrir la consola en la carpeta y ejecutar el comando “nodemon index.js”.
2.No es necesario crear la bd ya que está montada en mLab, con servidor web heroku.
3.Accedemos a la dirección localhost:3334/ la cual nos redirige al index.

Página montada en el servidor web:
https://ancode.herokuapp.com/
